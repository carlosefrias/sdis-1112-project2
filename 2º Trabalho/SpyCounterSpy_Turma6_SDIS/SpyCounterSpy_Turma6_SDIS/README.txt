Na pasta vai o projecto Eclipse SpyCounterSpyAndroidApp, o servidor AndroidGame feito no NetBeans e o código PHP da loja web.

Como não conseguimos colocar o servidor a correr numa máquina externa usamos o IP da máquina local; 
para puder testar a aplicação terá que criar o projecto AndroidGame no NetBeans, criar o projecto SpyCounterSpyAndroidApp no eclipse
e colocar no ficheiro RestClass.java na linha 44 o IP da sua máquina.
